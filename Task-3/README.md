# Task 3 – Web Application Security

This task demonstrates common web application vulnerabilities using DVWA.

Tools Used:
- DVWA (Damn Vulnerable Web Application)

Vulnerabilities Tested:
- SQL Injection
- Cross Site Scripting (XSS)
- Cross Site Request Forgery (CSRF)
- File Inclusion

Outcome:
Successfully demonstrated how attackers exploit web vulnerabilities and understood basic mitigation concepts.
